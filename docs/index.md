# Vilot docs

Vilot is an Obsidian desktop plugin that gives you an agentic chat assistant with vault tools, skill-based prompting, and safe multi-note editing.

## What Vilot gives you

- A persistent chat panel with streaming responses
- Vault-aware `@` context mentions for notes, folders, and tags
- A skill system based on `SKILL.md` definitions (built-in + user-installed)
- Tool transparency for each agent tool call (status, timing, result summary)
- Composer mode for reviewable multi-note edits before write
